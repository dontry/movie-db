# TV Shows DB

Demo: [https://tvshow-db.mccoy018.now.sh/](https://tvshow-db.mccoy018.now.sh/)

- Install and launch the app

  - `npm install`
  - `npm start`

- Testing

  - `npm run test`
  - `npm run storybook`

- Linting & format

  - `npm run lint`
  - `npm run format`

* Major libraries used:
  - Typescript
  - React
  - Redux
  - Redux-saga
  - Styled-components
  - Styled-system
  - Jest
  - Axios
